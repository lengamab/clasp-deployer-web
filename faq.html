ScriptFlow is a unified IDE and management dashboard for Google Apps Script and Make.com automation. It allows developers to build, deploy, and monitor workflows from a single, modern interface.

We offer a generous Free tier that includes essential features for individual developers. For teams and power users needing advanced integrations and higher usage limits, we offer Pro and Enterprise plans.

To use the Google Apps Script features, you just need a standard Google ID. However, for specialized Cloud Platform integrations (like BigQuery), a GCP project may be required.

ScriptFlow wraps the official Google CLASP CLI (Command Line Apps Script Projects). We handle the authentication and command execution in the background, giving you a GUI for pushing, pulling, and deploying code.
